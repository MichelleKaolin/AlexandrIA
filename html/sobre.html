Sobre o Projeto

O AlexandrIA nasceu dentro dos corredores da FATEC Osasco como uma resposta à dificuldade de centralizar e extrair conhecimento útil dos milhares de TCCs e artigos produzidos anualmente. A ideia foi criar mais do que um repositório: um ecossistema inteligente que "lê" o material acadêmico e auxilia o aluno através de Inteligência Artificial Generativa.

Ferramentas & Tecnologias

Para construir uma plataforma robusta e escalável, utilizamos o que há de mais moderno no desenvolvimento multiplataforma:

Java (Spring Boot) Python (FastAPI) LangChain (RAG) MySQL ChromaDB HTML5/CSS3 Custom Kotlin (Mobile Bridge)

Próximos Passos (Roadmap)

  • Implementação de transcrição de áudio para bancas de TCC.
  • Integração com a API da biblioteca física da FATEC.
  • Expansão do sistema de gamificação para recompensas físicas.
