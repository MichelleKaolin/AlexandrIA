# AlexandrIA 📚🤖

AlexandrIA is an intelligent academic platform designed to centralize, manage, and analyze undergraduate theses (TCCs). By leveraging an AI architecture based on RAG (Retrieval-Augmented Generation), the system allows students and researchers to query a real database, generating technical summaries and answering questions with zero hallucinations.

## 🚀 Features

- Smart Search: Find theses by course, technology, or topic.
- AI Engine (RAG): PDF parsing and automatic summary generation following academic standards.
- Gamification: Level and XP system (Beginner, Contributor, Specialist, Master).
- Community: Activity feed and contribution rankings.
- Secure Repository: Structured storage for academic documents.

## 🛠️ Tech Stack

### Front-end
- HTML5, CSS3 (Custom Navy & Gold Identity)
- Vanilla JavaScript (API Integration)

### Back-end
- **Java (Spring Boot):** User management, authentication, and security.
- **Python (FastAPI):** AI processing, LangChain, and LLM integration.

### Database & AI
- MySQL: Relational data persistence and activity logs.
- ChromaDB / Vector Store: Vector storage for semantic search.
- LangChain: AI workflow orchestration.
